# QA Júnior – Estudos e Prática

Este repositório contém exemplos de estudos em Quality Assurance,
focados em testes manuais para nível júnior.

## Conteúdo
- Casos de teste
- Cenários de teste
- Boas práticas de QA
- Estudos de testes funcionais

## Ferramentas
- Testes manuais
- Escrita de casos de teste
- Metodologia ágil (Scrum)

## Objetivo
Desenvolver habilidades práticas para atuação como QA Júnior.
